### 🫀 [Prédiction des Risques Cardiaques via Machine Learning](https://github.com/R7even31/ML_Cardiovascular-Risk-Prediction.git)
* **Objectif :** Modélisation du risque de maladie cardiaque en fonction de facteurs cliniques (cholestérol, âge, etc.).
* **Compétences :** Classification, Feature Engineering, Scikit-Learn, Évaluation de modèles médicaux (Recall, F1-Score).
* **Impact :** Développement d'un pipeline ML robuste optimisé pour minimiser les faux négatifs (critique en milieu médical).
